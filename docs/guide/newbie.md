# 新手指南

## 注册ourmc账号
1. 打开[ourmc皮肤站](https://skin.our-mc.cn)。      
2. 点击**现在注册**，输入您的真实**邮箱**。设置你的**密码**和**游戏内你所使用的用户名**，输入人机验证的验证码。     
3. 注册完成后，请前往你的邮箱，查找一封来自**ourmc服务器官方<team@our-mc.cn>**的邮件（如果没有请查看垃圾邮箱），若没收到，请回到网站上点击“再次发送”进行尝试。      ![再次发送](/再次邮件发送.jpeg)
4. 点击邮箱中的链接，再次输入一遍您的邮箱，以验证你所使用的邮箱是真实的。     
### 上传皮肤和使用
1. 打开[ourmc皮肤站](https://skin.our-mc.cn)，登录您的账号，来到用户中心，在侧边栏找到**皮肤库**。          
2. 您可以选择皮肤库中其他人的皮肤，也可以自己上传并使用。       
3. 在皮肤库界面里，点击上方导航栏中的**上传新皮肤**。       
4. 在这里上传皮肤设置即可。         
5. 上传完成后，在我的衣柜中找到您上传的皮肤，选择并使用即可。       

## 下载游戏
版本选择1.21.8，加载器可选Fabric。

## 配置启动器登录账户
### PCL2
选择你要使用的游戏版本（1.21.8的），点击**版本设置**-**设置**-**服务器**，按照如图填写
![PCL2](/PCL2.png)
> 认证服务器（必填）：`https://skin.our-mc.cn/api/yggdrasil`        
> 注册连接：`https://skin.our-mc.cn/auth/register`          
填写完成后**返回**即可（不要点击“设置为LittleSkin”），然后就可以登录您在我们皮肤站上注册的账户了！
### HMCL
**账户**-**添加认证服务器（外置登录）**     
服务器地址填写`https://skin.our-mc.cn/api/yggdrasil`，完成        
然后在侧边点击ourmc Skin，就可以登录您在我们皮肤站上注册的账户了！

## 进入服务器
1. 使用**1.21.8**的版本和**ourmc Skin**账号登录启动游戏。       
2. 进入**多人游戏**，**添加服务器**。     
3. 名称随意，服务器地址输入`mc.our-mc.cn`，完成。
4. 点击选择上刚刚添加的服务器，**加入服务器**。     
5. 首次进入需先同意我们的用户协议[EULA](/EULA)。
