# OURMC 服务器增值服务通用条款

**版本号：** 2.4.2A
**生效日期：** 2026年05月06日

本《OURMC服务器增值服务通用条款》（下称“本服务条款”）是《OURMC最终用户许可协议》（下称“主协议”）的补充组成部分。当您（下称“用户”）通过任何形式向OurMC服务器运营团队（下称“运营方”）支付费用以获得特定的游戏内权益、功能或虚拟物品时，即表示您已阅读、理解并同意完全接受本服务条款及主协议的全部内容约束。如您不同意本服务条款的任何内容，请勿进行任何支付行为。

### 第一条 总则与定义

1.1 本服务条款所述“增值服务”，特指用户在履行主协议义务的基础上，通过向运营方支付费用而额外获得的、由运营方提供的游戏内非必要性功能、虚拟权益、外观装饰及相关的技术支持服务。

1.2 增值服务的性质为基于用户自愿的**赞助支持**，用户所支付的费用是对服务器持续运营与发展的无偿捐助。运营方为答谢支持而提供的相应游戏内权益，是附条件的赠与回馈，不构成《中华人民共和国消费者权益保护法》所定义的普通商品或服务买卖关系。

### 第二条 服务内容与运营方权利

2.1 运营方有权根据技术条件、游戏版本、运营策略及社区反馈，独立决定、设计、提供、调整、暂停或终止各项增值服务的内容、名称、具体权益及呈现方式。具体的服务目录、权益描述及价格将以运营方在官方渠道（包括但不限于官方网站、游戏内公告、社群公告）另行公布的清单或说明为准。

2.2 运营方承诺，所有增值服务的设计与提供将严格遵守主协议**第二章第2条**的原则，确保不会通过出售直接影响游戏角色能力数值、属性强度、竞争公平性的项目来破坏游戏平衡。

2.3 运营方保留对增值服务的技术实现方式、后台系统进行升级、维护、迁移或重置的权利。在必要时，运营方有权以功能与价值近似的其他权益，替换原有的某项具体权益。

### 第三条 用户义务与使用规范

3.1 用户在购买及使用增值服务时，除需遵守主协议的全部规定外，还需特别遵守以下规定：
    (a) 不得将所获得的增值服务权益（包括但不限于账号身份、特殊权限、虚拟物品）用于任何商业目的，包括但不限于转让、出租、出售、代练或用于置换现实货币及其他虚拟财产。
    (b) 不得利用增值服务所附带的任何功能或便利，实施干扰、破坏其他用户正常游戏体验、损害服务器稳定或违反游戏公平性的行为。
    (c) 不得对与增值服务相关的任何系统、代码、数据进行破解、反编译、修改或制作任何第三方辅助程序。

3.2 用户理解并同意，其所获得的增值服务权益与特定的游戏账号（以Mojang/Microsoft正版UUID为准）绑定。该权益仅限于该账号持有者本人在OURMC服务器内使用，禁止任何形式的账号共享、出借或交易。因用户违反此款规定导致的权益损失，由用户自行承担。

### 第四条 支付、账户与退款政策

4.1 用户应通过运营方官方指定且公告的支付渠道完成费用支付。任何通过非官方渠道（如第三方代充、非授权平台交易）进行的支付行为均不被认可，且可能导致用户账号受到处罚，运营方不承担因此造成的任何损失。

4.2 用户在支付前，有义务仔细核对所选择的增值服务项目、金额及对应的账号信息。支付行为一经完成，即视为订单确认。

4.3 **退款政策**：基于增值服务的数字商品特性及赞助支持性质，用户一旦支付成功，除非因运营方原因导致该服务完全无法提供且无替代方案，否则**运营方不提供任何形式的退款、撤销或权益折现服务**。此政策明确包括但不限于以下情况：
    (a) 用户个人原因导致的误购、改变主意或不再游玩；
    (b) 用户因违反主协议或本服务条款导致账号被封禁、服务权限被终止；
    (c) 服务器因正常维护、版本更新、网络波动等原因导致的临时性服务中断。

### 第五条 免责与责任限制

5.1 用户理解并同意，运营方将尽合理努力保障增值服务的稳定提供，但不对其提供任何明示或默示的持续性、可用性、及时性、安全性或适用性保证。

5.2 因下列原因导致用户无法使用增值服务或相关数据丢失的，运营方不承担责任：
    (a) 服务器例行维护、升级、迁移或遭遇不可抗力（如自然灾害、政府行为、骨干网络中断）；
    (b) 用户本地网络、设备或客户端问题；
    (c) 用户操作失误或账号安全问题；
    (d) 游戏版本重大更新导致部分增值服务功能需进行技术调整或暂时无法兼容。

5.3 在任何情况下，运营方就增值服务所承担的全部责任，无论基于何种原因，均不应超过用户就该项服务所实际支付的费用金额。

### 第六条 协议变更与终止

6.1 运营方有权在必要时单方修改本服务条款，修改后的条款将在官方渠道提前公告。若用户在新条款生效后继续使用增值服务，即视为接受修改；若不同意，用户有权停止使用相关服务。

6.2 在以下情况下，运营方有权单方终止向用户提供全部或部分增值服务，且无需承担任何责任：
    (a) 用户严重违反主协议或本服务条款；
    (b) 应法律法规、政策要求或有权机关的命令；
    (c) 出于运营方整体业务调整、技术升级或安全考虑。

6.3 本服务条款终止后，其中关于知识产权、免责声明、法律适用与争议解决的条款，以及根据其性质应持续有效的条款，将继续对双方具有约束力。

### 第七条 法律适用与争议解决

7.1 本服务条款的订立、效力、解释、履行及争议的解决，均适用中华人民共和国大陆地区法律。

7.2 因本服务条款引起的或与之相关的任何争议，双方应首先友好协商解决；协商不成的，任何一方均有权将争议提交至**运营方所在地有管辖权的人民法院**通过诉讼解决。
